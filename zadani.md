Webové stránky budou obsahovat tyto náležitosti:

budou tvořeny pomocí jazyka HTML5 a kaskádových stylů CSS, primárně nebude hodnocen vzhled webových stránek, ale jejich obsah, struktura obsahu, vyplnění požadovaných HTML značek a další vzhledem k SEO optimalizaci a optimalizaci rychlosti načítání webu
součástí webových stránek bude hlavní nadpis nebo logo, navigační menu, obsahová část a patička obsahující kontaktní a další informace
dále budou součástí webových stránek alespoň 2 CSS styly (hlavní CSS styl - např. CSS frameworku, custom styl), alespoň 2 JS soubory (např. jQuery, JS pro calendar picker, fotogalerii, apod.)
hlavní strana bude obsahovat 5 obrázků
součástí webových stránek bude sekce Fotogalerie, která bude obsahovat alespoň 20 obrázků ve dvou velikostech (náhledy + velké obrázky - velikost min. 800px)
webové stránky budou obsahovat alespoň 5 vzájemně provázaných (pomocí navigačního menu) HTML souborů, přičemž každý HTML soubor reprezentuje jednu podstránku (např. O nás, Služby, Produkty, Kontakt, apod.)
v každém HTML souboru budou správně vyplněny značky <title>, <h1>, <h2>, <h3> (případně další úrovně pokud je potřeba), meta značky a dobře strukturovaný obsah tvořený pomocí zásad copywritingu - v obsahu se budou ve vhodném poměru vyskytovat klíčová slova a fráze, které jste si pro projekt zvolili (pro SEO optimalizaci při vyhledávání pomocí internetového vyhledávače)
každý HTML soubor bude pojmenován SEO-friendly způsobem
jednotlivé HTML soubory (stránky) doplňte o vhodné URL odkazy na další podstránky Vašeho webu (např. v text, v obsahu stránky) mimo navigační menu
webové stránky poté nahrajte na libovolný webhosting
na webové stránky aplikujte optimalizaci velikosti kódu (minimálně minifikace, případně caching, apod.), techniku Lazy loading pro načítání obrázků, optimalizaci rychlosti načítání webu - co nejnižší hodnota metriky First Contentful Paint (CFP)
webové stránky propojte s nástrojem Google Search Console, dále doplňte soubor robots.txt do kořenového adresáře webu a přidejte vygenerovanou mapu stránek ve formě sitemap.xml, dále aktivujte indexaci webových stránek v nástroji Google Search Console
po dobu cca 10-14 dní sledujte, zda se web zaindexoval do internetového vyhledávače Google a výsledky vyhledávání Vašeho webu na Vámi zvolená klíčová slova zaznamenejte do PDF souboru (popis viz níže)
webové stránky propojte se nástrojem pro sledování návštěvnosti Google Analytics a zajímavé statistiky o návštěvnosti a s tím spojených datech zaznamenejte do PDF souboru
webové stránky budou striktně HTML5 validní
